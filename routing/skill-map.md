# Skill Map

Updated: 2026-05-26

Use when: A task names or implies a reusable workflow pattern.

## Direct Invocations

- Algorithm, ISC, audit discipline, frozen handoff ->
  `skills/algorithm/SKILL.md`
- ISA, audit artifact, structured handoff -> `skills/isa/SKILL.md`
- Memory, remember, prior context, distill -> `skills/memory/SKILL.md`

## Intent Routes

- Code review, PR review, regression risk -> `skills/code-review/SKILL.md`
- Evidence-first implementation, high-risk fix ->
  `skills/verifier-first/SKILL.md`
- Current facts, external docs, source-backed answer ->
  `skills/research/SKILL.md`
- Multi-agent audit-bearing work -> recommend `skills/algorithm/SKILL.md`;
  use it only after explicit invocation or accepted recommendation
- Durable audit artifact needed -> `skills/isa/SKILL.md`

## Use Rules

Load only the skill files needed for the turn. Skills may compose when their
responsibilities are distinct.

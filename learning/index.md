# Learning Index

Updated: 2026-05-26

Use when: A task creates reusable lessons, asks to remember a correction, or
calls for distillation.

Learning is manual-first. Raw notes go to `inbox.md`; periodic reviews promote
durable knowledge into memory, skills, or routing.

## Files

- `inbox.md` captures raw reusable lessons.
- `distillations/` stores cleaned knowledge promoted from the inbox.
- `reviews/` records periodic cleanup and promotion decisions.
- `prompts/` stores reusable prompts for learning review sessions.

## Capture Threshold

Capture named triggers: new decision, repeated workflow, correction received,
or a plainly reusable signal. Avoid logging ordinary task chatter.

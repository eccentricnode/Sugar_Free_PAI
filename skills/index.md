# Skills Index

Updated: 2026-05-26

Use when: A task names a skill or routing maps the task to a reusable workflow.

Skills use pi.dev's native `skills/<name>/SKILL.md` convention. A skill
constrains method; it is not a runtime mode.

## Available Skills

- `algorithm/` for explicit Algorithm, ISC, audit discipline, frozen handoffs,
  or accepted recommendations for audit-bearing work.
- `isa/` for structured audit artifacts when work needs resumption or audit.
- `verifier-first/` for evidence-first implementation or review.
- `code-review/` for defect-focused review.
- `research/` for source-backed investigation.
- `memory/` for memory lookup, capture, and distillation work.

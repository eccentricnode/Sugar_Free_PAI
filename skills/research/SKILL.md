# Research

## When To Use

Use when the task needs current facts, external documentation, source
comparison, or precise attribution.

## Operating Rules

- Prefer primary sources.
- Record dates when recency matters.
- Distinguish sourced facts from inference.
- Keep copied text short and summarize in your own words.

## Workflow

1. Define the question and freshness requirement.
2. Search or read the smallest source set that can answer it.
3. Compare conflicting sources when needed.
4. Answer with citations and explicit uncertainty where appropriate.

## Output Expectations

Provide links or file references for the sources used.
